# Proyecto-4
Análisis exploratorio de datos de compras en plataforma de e-commerce
En este proyecto realicé un análisis exploratorio de datos utilizando un conjunto de datos de Instacart, una plataforma de entrega de comestibles en línea. El objetivo principal fue comprender los hábitos de compra de los usuarios y generar insights que puedan mejorar la experiencia del cliente y la toma de decisiones comerciales.

El trabajo inició con la integración y preprocesamiento de múltiples tablas relacionadas (pedidos, productos, categorías y departamentos), donde se abordaron problemas de calidad de datos como valores ausentes, duplicados y tipos de datos incorrectos. Este proceso permitió construir un dataset limpio y consistente para el análisis.

Posteriormente, llevé a cabo un análisis exploratorio enfocado en patrones de comportamiento de compra. Se analizaron variables clave como la hora del día y el día de la semana en que los usuarios realizan pedidos, así como el tiempo entre compras. A través de visualizaciones, identifiqué tendencias en la actividad de los usuarios y variaciones en sus hábitos de consumo.

Además, se investigaron métricas relevantes como la frecuencia de pedidos por cliente, los productos más populares y aquellos con mayor tasa de recompra. También se analizó el comportamiento dentro del carrito de compra, identificando qué productos suelen añadirse primero, lo que aporta información sobre prioridades y patrones de decisión del usuario.

Los resultados permitieron identificar tendencias clave en el comportamiento de compra, como la recurrencia de ciertos productos, patrones temporales en la demanda y diferencias en la fidelidad del cliente, proporcionando una base sólida para estrategias de marketing y optimización de la experiencia de usuario.

Este proyecto demuestra habilidades en limpieza de datos, integración de múltiples fuentes, análisis exploratorio, visualización de datos y generación de insights orientados a negocio dentro del contexto de comercio electrónico.
